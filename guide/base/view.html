# 视图

采用 Twig 本身作为模板引擎。

# 模板目录

模板目录置于public/views，原则上分模块以及分控制器进行新建模板

在控制器controller中,使用 $this->render('模板路径',compact('变量'))，如：

     public function index()
     {
         $name = 'KickPeach';
         return $this->render('/Home/index.html',compact('name'));
     }

变量赋值可使用compact进行变量的传递。
